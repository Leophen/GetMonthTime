# LpTemplate 初始化组件

用于作为其他组件提供模板

## API

### Props

| 参数          | 说明                                                    | 类型    | 可选值 | 默认值 |
| ------------- | ------------------------------------------------------- | ------- | ------ | ------ |
| value/v-model | 是否显示浮动面板                                        | boolean | -      | -      |

### Slots

| name    | 说明                     |
| ------- | ------------------------ |
| header  | 用于自定义浮动面板的首部 |
| content | 用于自定义浮动面板的内容 |
